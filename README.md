# 项目概述 python424

本项目是一个基于Python的音乐推荐系统，结合了可视化分析功能。用户可以通过登录使用系统，实现对歌曲的协同过滤推荐，并能基于歌单进行可视化分析。

# 技术栈

- 后端：Flask
- 前端：Vue
- 语言与工具：Python
- 算法：协同过滤算法
- 可视化：ECharts
